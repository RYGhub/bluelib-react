A **panel** which is rendered like a `<table>` element.

```jsx
import Bluelib from "../Bluelib";

<Bluelib>
    <Table>
        <thead>
            <tr>
                <th>Column 1</th>
                <th>Column 2</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>A</td>
                <td>B</td>
            </tr>
            <tr>
                <td>C</td>
                <td>D</td>
            </tr>
        </tbody>
    </Table>
</Bluelib>
```
